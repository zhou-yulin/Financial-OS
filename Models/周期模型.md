# 周期模型

## 概述

周期模型回答：**我们当前处于什么位置？周期如何影响决策？**

## 一、周期的三层结构

### 1.1 三层周期叠加

```
心理周期（最短，6个月 - 3年）
    │
    ▼
市场周期（中期，3 - 7年）
    │
    ▼
经济周期（最长，5 - 10年）
```

所有的市场判断都需要考虑这三层周期的叠加效果。

### 1.2 三层周期对比

| 周期 | 驱动力 | 持续时间 | 可预测性 | 如何判断 |
|------|--------|----------|----------|----------|
| 经济 | GDP、就业、通胀 | 5-10年 | 中 | PMI、利率、失业率 |
| 市场 | 估值、流动性、情绪 | 3-7年 | 低中 | PE水平、CAPE、信用利差 |
| 心理 | 恐惧/贪婪 | 6月-3年 | 可预测（人性不变） | 媒体、身边人讨论频率 |

## 二、周期判断指标

### 2.1 经济周期指标

```
扩张期信号（经济上行）：
- GDP增速 > 趋势线
- PMI > 50
- 失业率下降
- 消费者信心上升
- 信贷增长

收缩期信号（经济下行）：
- GDP增速 < 趋势线
- PMI < 50
- 失业率上升
- 消费者信心下降
- 信贷收紧
```

### 2.2 市场周期指标

```
股市估值：
- PE 处于历史低 20% → 可能被低估
- PE 处于历史高 20% → 可能被高估
- 股债收益比 > 2 → 股票相对便宜
- 股债收益比 < 1 → 股票相对贵

信用市场：
- 信用利差扩大 → 风险厌恶上升
- 信用利差收窄 → 风险偏好上升
```

### 2.3 心理周期指标（情绪温度计）

```
极度恐惧（可能接近底部）：
□ 媒体充满悲观报道
□ 所有人都在说"现金为王"
□ 投资群安静或充满哀嚎
□ 新手投资者纷纷割肉离场

极度贪婪（可能接近顶部）：
□ 媒体大量报道暴富故事
□ 从不投资的人开始问"买什么好"
□ IPO激增，质量下降
□ 杠杆使用普遍
□ "这次不一样"成为流行语
```

## 三、在周期中的行动策略

### 3.1 根据周期位置调整策略

| 周期位置 | 核心策略 | 仓位建议 |
|----------|----------|----------|
| 极度恐惧 | 积极买入优质资产 | 重仓 >80% |
| 谨慎恢复 | 维持仓位，选择性加仓 | 70-80% |
| 正常时期 | 正常配置，定投 | 标准配置 |
| 极度贪婪 | 逐步减仓，提高现金 | <60% |

### 3.2 不要试图精确择时

周期判断 = "知道现在的风险收益比"。不是"知道明天会涨还是跌"。

**精确择时 = 不可能。周期位置判断 = 可能且有用。**

### 3.3 定投穿越周期

如果你不确定周期位置，定投是最优策略。它自动在高位少买、低位多买，无需你的判断。
