# 信号与噪音模型

## 定义

**信号 = 能帮助你做出更好决策的有用信息。**
**噪音 = 浪费你时间和注意力、让你做出更差决策的干扰信息。**

在投资和商业中，区分信号与噪音是最高级的技能之一。

## 一、信号的特征

- **持久性**：6个月后仍然重要
- **可验证性**：可以从多个独立来源交叉验证
- **因果关系**：有明确的因果链条（因为A所以B）
- **量级匹配**：信息的重要性与你的决策规模匹配

## 二、噪音的特征

- **随机性**：前后不连贯、没有模式
- **情绪驱动**：激发恐惧或贪婪
- **短期性**：24小时后就不重要了
- **无法验证**：来源单一、无法追溯
- **细节过多**：被大量无关细节淹没

## 三、信息过滤器框架

在接收任何信息之前，通过三个测试：

### 3.1 6个月测试

```
六个月后，这条信息还有价值吗？
├── 是 → 可能是信号
└── 否 → 大概率是噪音
```

### 3.2 可行动性测试

```
这条信息能让我做不同的决策吗？
├── 是 → 值得关注
└── 否 → 忽略（无论它多引人注目）
```

### 3.3 相关性测试

```
这条信息与我关注的企业/行业直接相关吗？
├── 是 → 分析
└── 否 → 存档或忽略
```

## 四、在投资中的应用

| 信号（值得关注） | 噪音（忽略） |
|-----------------|-------------|
| 护城河变化 | 每日股价波动 |
| 管理层更替 | 分析师评级变更 |
| 长期毛利率趋势 | 单季EPS波动 |
| 行业竞争格局变化 | 股市每日涨跌评论 |
| 客户续约率变化 | 政客的政策话题 |
| 商业模式变化 | "下一个牛股"推荐 |

## 五、噪音的陷阱

- **信息成瘾**：不断刷新闻让你感觉在"工作"，但实际是在浪费精力
- **确认偏误**：只吸收支持已有判断的信息
- **叙事错觉**：把随机的市场波动变成"有道理的故事"
- **过度反应**：对短期事件做出过于长期的结论

**对付噪音最有效的方法不是更好的分析，而是减少摄入。**
