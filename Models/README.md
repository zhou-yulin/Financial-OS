# 模型交叉引用索引

> 本文件是 Models/ 目录下所有思维模型的交叉引用索引。每个模型按类别分组，标注其对应的操作系统（OS）和关联案例。

---

## 模型分组总览

| 分组 | 包含模型 | 核心用途 |
|------|----------|----------|
| 🧱 **基础模型** | 财富模型、企业模型、周期模型 | 理解世界运作的本质规律 |
| 🏢 **商业模型** | 竞争模型、网络效应、飞轮模型、S-Curve | 分析商业竞争与增长逻辑 |
| 📈 **投资模型** | 估值模型、信号与噪音模型、机会成本 | 辅助资金配置与投资决策 |
| 🧠 **认知模型** | 反馈循环、创新模型 | 提升思维质量与创造力 |

## 模型引用规范

在 OS 和 Book 文件中引用模型时的标准格式：

```
参考 `Models/模型名称.md`
```

示例：

```
- 参考 `Models/估值模型.md` 进行价格判断
- 参考 `Models/网络效应.md` 分析企业护城河
- 参考 `Models/飞轮模型.md` 理解增长的自我加速机制
```
