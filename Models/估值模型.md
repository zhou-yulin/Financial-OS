# 估值模型

## 概述

估值模型回答：**这个东西值多少钱？现在买划算吗？**

## 一、估值核心原理

### 1.1 基本原理

**任何资产的价值 = 它未来所有现金流的现值。**

这是所有估值方法的理论基础。问题是如何估计"未来的现金流"。

### 1.2 估值方法选择

| 方法 | 公式 | 最适合 | 最不适合 |
|------|------|--------|----------|
| DCF | V = Σ(CF_t/(1+r)^t) | 稳定现金流公司 | 高增长不确定公司 |
| PE | 股价/EPS | 盈利稳定的公司 | 周期股、亏损公司 |
| PB | 股价/净资产 | 金融、银行 | 轻资产公司 |
| PS | 股价/营收 | 亏损成长公司 | 高利润公司（应看PE） |
| 反向DCF | 从市值反推增长率 | 所有公司 | 了解市场预期 |
| PEG | PE/增长率 | 成长型公司 | 负增长公司 |

## 二、DCF 实战案例

### 2.1 案例：某成熟消费品公司

假设我们要估算一家成熟的消费品公司。数据如下：

| 指标 | 数值 |
|------|------|
| 最新自由现金流 | 50 亿元 |
| 营收增速 | 5% |
| 利润率趋势 | 稳定 |
| 资本支出 | 稳定 |
| 净负债 | 100 亿元 |
| 发行在外股份 | 20 亿股 |
| 当前股价 | 80 元 |

### 2.2 估值假设

| 假设项 | 悲观 | 基本 | 乐观 |
|--------|------|------|------|
| 现金流增速 | 3% | 5% | 7% |
| 永续增长率 | 2% | 3% | 4% |
| 折现率 WACC | 12% | 10% | 8% |

### 2.3 估算步骤

**第一步：预测自由现金流**

假设基本情景（增速 5%）：

```
第 1 年：50 × (1+5%) = 52.5 亿
第 2 年：52.5 × (1+5%) = 55.1 亿
第 3 年：55.1 × (1+5%) = 57.9 亿
第 4 年：57.9 × (1+5%) = 60.8 亿
第 5 年：60.8 × (1+5%) = 63.8 亿
```

**第二步：快速估算法**

```
快速内在价值 ≈ 自由现金流 × (1 + g) / (r - g)

= 50 × (1 + 3%) / (10% - 3%)
= 51.5 / 0.07
= 735.7 亿（企业价值）
```

**第三步：调整到每股价值**

```
企业价值：735.7 亿
减去净负债：100 亿
股权价值：635.7 亿

每股价值 = 635.7 / 20 = 31.79 元

当前股价：80 元
安全边际：31.79 / 80 = 0.40（不安全——股价远高于估值）
```

**结论**：当前股价 80 元 vs 内在价值 32 元 → 高估。但如果你对增长有更乐观的假设，结论会不同。关键不在于精确的数字，而在于**差距是否大到可以决策**。

### 2.4 分阶段 DCF（高增长 → 稳定）

许多公司不是匀速增长，而是前期高增长、后期稳定增长。此时使用两阶段 DCF：

```
内在价值 = Σ(CF_t / (1+r)^t) + [CF_n × (1+gn) / (r - gn)] / (1+r)^n
           └──────── 增长期 ──────┘   └──────── 终值 ──────────┘
```

### 2.5 DCF 检查清单

```
- [ ] 自由现金流是否真实反映盈利能力？（检查：经营现金 vs 净利）
- [ ] 增长率假设是否合理？（不能长期高于 GDP 的 2-3 倍）
- [ ] 折现率是否反映真实风险？（股权成本 = 无风险利率 + β × 股权风险溢价）
- [ ] 终值占比是否合理？（如果 > 80%，估值几乎完全依赖假设）
- [ ] 多情景分析是否做了？（给一个范围，而不是一个数字）
```

## 三、反向 DCF 自动化

### 3.1 反向 DCF 的逻辑

反向 DCF 不问你"假设什么增长率"，而是问你"当前市值隐含了什么增长率"。

操作步骤：

```
当前市值 → 反推未来自由现金流的复合增长率
```

### 3.2 反向 DCF 计算模板

```
## 输入
当前市值：________ 亿元
最新自由现金流：________ 亿元
折现率（通常 10%）：________%
预测期：10 年
终值增长假设：3%

## 计算步骤
1. 假设未来 10 年现金流年增长率 = x%
2. 计算 10 年现金流现值之和
3. 计算终值现值
4. 企业价值 = 2 + 3
5. 减去净负债 = 股权价值
6. 解出 x（使股权价值 = 当前市值）

## 对照
- 如果 x > 行业历史增速 → 市场预期偏高
- 如果 x < 行业历史增速 → 可能被低估
- 如果 x > 20% → 市场对完美执行的极高预期（高风险）
```

### 3.3 快速判断：PE + 增长

```
合理的 PEG（PE / 增速）范围：

PEG < 0.5 → 显著低估
PEG 0.5-1.0 → 合理偏低估
PEG 1.0-1.5 → 合理偏高估
PEG > 1.5 → 高估

例：PE 25 倍，增长率 20% → PEG = 25/20 = 1.25 → 合理偏贵
     PE 15 倍，增长率 15% → PEG = 15/15 = 1.0 → 合理
```

### 3.4 三分钟估值速算

```
快速判断公司的估值水平：

1. PE 倍数 ÷ (增长率 × 100) = PEG
2. 如果 PEG < 1 → 看看是否真的低估
3. 如果 PEG > 1.5 → 看看是否有特殊理由

注意：PEG 是快速过滤工具，不是精确估值方法。
```

## 四、综合估值框架

### 4.1 不同公司的估值重点

| 公司类型 | 核心指标 | 重点关注 | 警示信号 |
|----------|----------|----------|----------|
| 成熟消费品 | PE, DCF | 现金流稳定性 | 毛利率下降 |
| 成长科技 | PS, 反向DCF | 市场份额、增长持续性 | 获客成本上升 |
| 金融公司 | PB, ROE | 资产质量、杠杆率 | 不良贷款上升 |
| 周期股 | 平均利润 PE | 周期性容量的位置 | 在顶部用 PE |
| 互联网平台 | 用户数 × 用户价值 | 网络效应强度 | 用户增长放缓 |
