# 行为金融模型

## 定义

行为金融学（Behavioral Finance）是心理学与金融学的交叉学科，研究投资者在决策过程中因认知偏误和情绪影响而产生的**非理性行为**，以及这些行为如何系统性地影响资产定价和市场价格的形成。

---

## 一、行为金融 vs 传统金融对比

| 维度 | 传统金融（标准金融学） | 行为金融 |
|------|----------------------|----------|
| **核心假设** | **理性人假设**：投资者完全理性，能无偏处理所有信息 | **有限理性**：投资者受认知局限和情绪影响，决策存在系统性偏差 |
| **市场有效性** | **有效市场假说**：价格充分反映所有可得信息，无套利机会 | **市场异象存在**：因非理性行为，价格经常偏离内在价值 |
| **决策模型** | 期望效用理论：投资者追求风险调整后效用最大化 | 前景理论：对收益风险厌恶，对损失更强烈厌恶；参考点依赖 |
| **方法论** | 规范性的（应如何决策） | 描述性的（实际如何决策） |
| **风险理解** | 波动率 = 风险，用 CAPM β 衡量 | 风险感知受认知偏误扭曲，存在行为β |
| **套利** | 套利者立即消除错误定价 | **有限套利**：噪音、成本、期限限制阻止套利 |

### 理性人假设 vs 有限理性

- **理性人假设**：人们利用所有可用信息，按贝叶斯法则更新信念，做出期望效用最大化的选择。
- **有限理性**：认知能力有限，使用经验法则（启发式）简化决策，导致可预测的偏误。

### 有效市场 vs 市场异象

- **有效市场**：价格随机游走，技术分析无效，主动管理无法长期跑赢市场。
- **市场异象**：大量实证研究发现可预测的价格模式，无法用传统风险因子解释。

## 二、12 大关键行为偏误概述

| # | 偏误名称 | 定义 | 投资影响 |
|---|---------|------|---------|
| 1 | **确认偏误** | 倾向于寻找、注意和相信支持自己已有观点的信息 | 忽视反面证据，过度持有亏损头寸 |
| 2 | **锚定效应** | 决策时过度依赖最先获得的信息（"锚"） | 以买入价为锚，不愿在合理价位止损 |
| 3 | **损失厌恶** | 对损失的敏感度远高于同等收益（约 2-2.5 倍） | 过早卖出盈利股、死守亏损股（处置效应） |
| 4 | **过度自信** | 高估自己的知识、能力或信息精确度 | 过度交易，交易频率越高收益越低（Barber & Odean 经典结论） |
| 5 | **羊群效应** | 模仿他人决策，忽视自身私有信息 | 追涨杀跌，加剧泡沫与崩盘 |
| 6 | **近因偏误** | 过分看重近期事件，忽视长期趋势 | 追热点、趋势末端入场 |
| 7 | **可得性启发** | 依据容易想到的例子的频率判断事件概率 | 高估罕见事件（如股灾）概率，低估常态风险 |
| 8 | **自我归因** | 将成功归因于自己能力，将失败归因于外部因素 | 不吸取教训，反复犯同样错误 |
| 9 | **后见之明** | 事件发生后认为"我早就知道" | 过度自信增倍，削弱复盘学习效果 |
| 10 | **沉没成本谬误** | 因已投入资源而继续坚持亏损决策 | 越跌越补，拒绝认错离场 |
| 11 | **熟悉偏误** | 偏好投资自己熟悉或本土的公司/行业 | 缺乏分散化，承担不必要的非系统性风险 |
| 12 | **心理账户** | 将资金按来源或用途划分到不同心理账户 | 对不同资金采用不同风险标准，整体组合非最优 |

## 三、市场异象

### 1. 一月效应

- 小盘股在一月份（尤其前两周）出现异常高收益。
- 解释：年末税务亏损卖出 + 新年重新建仓 + 窗口粉饰后回补。

### 2. 动量效应

- 过去 3-12 个月表现好的股票（赢家）在未来继续跑赢过去表现差的股票（输家）。
- Jegadeesh & Titman (1993) 经典实证：买入赢家、卖出输家的组合年化超额收益约 12%。
- 行为解释：**反应不足** + **羊群效应** + **确认偏误**。

### 3. 价值效应

- 低 PE、低 PB、高股息率等价值股长期跑赢成长股。
- Fama-French 三因子模型将价值溢价归为风险因子，但行为金融认为来自**过度外推**（投资者对成长股过度乐观）。

### 4. IPO 折价

- 新股上市首日平均收益率显著为正（美国约 15-20%，中国更高）。
- 行为解释：**发行人为避免诉讼故意折价** + **投资者狂热推高首日价格** + **赢者诅咒**。

### 5. 泡沫形成与破裂

| 阶段 | 投资者心理 | 偏误驱动 |
|------|-----------|---------|
| 萌芽期 | 新叙事出现，早期采纳者入场 | 锚定（初始价格低） |
| 繁荣期 | 价格持续上涨，媒体关注 | 确认偏误 + 羊群效应 |
| 狂热期 | 全民参与，杠杆增加 | 过度自信 + 可得性启发 |
| 破裂期 | 价格暴跌，恐慌性抛售 | 损失厌恶 + 羊群效应（反向） |

## 四、行为金融在投资中的应用

### 1. 利用偏误识别定价错误

| 偏误 | 可识别的定价错误 | 策略 |
|------|----------------|------|
| 过度外推 | 成长股被高估，价值股被低估 | 价值投资（深度价值） |
| 处置效应 | 亏损股被超卖，盈利股被超买 | 反向交易 |
| 羊群效应 | 热门股超买，冷门股超卖 | 反向交易 |
| 近因偏误 | 趋势末端价格偏离 | 趋势 + 均值回归结合 |
| 注意力驱动 | 新闻热点股过度反应 | 事件驱动 + 反转 |

### 2. 建立系统克服自身偏误

- **强制决策流程**：制定严格的买入/卖出/仓位管理规则，减少临时判断。
- **投资清单**：每次交易前检核偏误检查清单。
- **独立研究**：不依赖分析师共识，建立自己的独立观点。
- **分散化强制**：设定行业/个股仓位上限，避免熟悉偏误。
- **复盘机制**：记录每笔交易及心理状态，客观评估决策过程而非结果。
- **团队决策**：不同观点的碰撞减少确认偏误。

## 五、结合 Decision-OS 的偏误防护清单

> 以下清单可嵌入 Decision-OS 的决策流程中，在每次买入/卖出/持仓评估前自动触发。

### 🔴 交易前检核清单

```
[ ] 买入理由是否有数据支持？还是只是"感觉"？（对抗过度自信）
[ ] 是否存在与我仓位方向相反的证据？（对抗确认偏误）
[ ] 参考价格设定是否有客观依据？还是被买入价锚定？（对抗锚定效应）
[ ] 若现在持有现金，是否仍会买入该资产？（对抗沉没成本谬误）
[ ] 持仓比例是否超出分散化规定的上限？（对抗熟悉偏误）
[ ] 决策受近期行情影响多大？是否考虑过 3-5 年周期数据？（对抗近因偏误）
[ ] 这笔交易是否受他人影响？是否做了独立分析？（对抗羊群效应）
[ ] 对该资产收益率的预期是否基于容易回忆的案例？（对抗可得性启发）
```

### 🟡 持仓期检核清单

```
[ ] 是否因为亏损而拒绝止损？（损失厌恶 + 沉没成本）
[ ] 近期盈利是否导致信心过度膨胀？（自我归因 + 过度自信）
[ ] 是否有新信息被忽视？为什么？（确认偏误）
[ ] 是否在不同的心理账户中使用了不同的风险标准？（心理账户）
```

### 🟢 复盘检核清单

```
[ ] 结果好坏 = 决策质量好坏？（避免结果偏误）
[ ] 在结果出来之前，决策本身是否合理？（好的决策可能产生坏结果）
[ ] 如果结果是好的，是我做对了什么还是运气？（对抗自我归因）
[ ] 如果结果是坏的，是我做错了什么还是运气？（避免后见之明）
[ ] 从这笔交易中学到了什么可以系统化的规则？
```

---

> 行为金融学的核心洞见：**市场并非总是理性的，但偏误是可预测的。认识偏误、系统化防护，是长期超额收益的重要来源。**
