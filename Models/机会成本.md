# 机会成本模型

## 定义

**机会成本（Opportunity Cost）= 当你选择 A 时，自动放弃的次优选项 B 的价值。**

任何选择都有代价。代价不是你付出多少钱，而是你放弃了什么。

## 一、机会成本的本质

### 1.1 经济学基础

- 每种资源（时间、金钱、注意力）的每一种用途，都在阻止它被用于其他用途
- 真正的成本不是账面上的数字，而是**你放弃的最高价值替代方案**

### 1.2 为什么机会成本被忽视？

1. **显性 vs 隐性**：显性成本（花了多少钱）很容易看到，隐性成本（放弃了什么）不容易看到
2. **即时 vs 延迟**：当前决策的收益是即时的，放弃的替代方案的收益是延迟的
3. **确定性 vs 不确定性**：你拥有的机会看起来是确定的，你放弃的机会是不确定的

**大脑天然倾向于选择此刻确定的东西，哪怕长期来看并不划算。**

## 二、在投资中的应用

### 2.1 持仓的隐性成本

持有一只股票的成本不只是买入价格，还包括**你本可以用这笔钱买别的**。

```
选择 A：持有腾讯（当前收益 0%）
选择 B：换成 Meta（预期收益 15%）

机会成本 = 15%
```

**亏钱不是唯一的成本。不去赚本该能赚的也是。**

### 2.2 闲置现金的成本

持有现金看起来"安全"，但现金的机会成本是：

- 通胀侵蚀购买力（年约 3-5%）
- 错失市场长期增长（年约 8-10%）
- 被税收侵蚀（通胀税）

当然，现金也有期权价值——在危机中你有子弹。但这需要权衡。

### 2.3 如何将机会成本纳入决策

每次投资决策时，问自己：

```
1. 如果我没持有这笔资产，我现在会买入吗？
2. 我放弃的第二佳选择是什么？它的预期回报是多少？
3. 我为何认为选择 A 大于选择 B？
```

## 三、在生活中的应用

### 3.1 时间的成本

**时间是你最稀缺的资源。**

- 刷 2 小时短视频的机会成本 = 读完一本书的 20-30 页
- 在低价值工作上加班的机会成本 = 学习新技能的时间
- 参加一个无聊聚会的成本 = 陪伴家人或深度休息的机会

**你每选择做一件事，同时选择了不做所有其他事。**

### 3.2 机会成本随年龄变化

| 年龄 | 最珍惜的资源 | 机会成本最高的活动 |
|------|-------------|-------------------|
| 20-30 | 时间 × 学习能力 | 做没有成长性的工作 |
| 30-45 | 时间 × 精力 | 分散注意力于太多方向 |
| 45+ | 时间 × 健康 | 牺牲健康追逐金钱 |

### 3.3 机会成本 vs 沉没成本

**沉没成本**：已经花出去、无法收回的成本（不要考虑它）。

**机会成本**：你的选择所放弃的其他选择的价值（这是你应该考虑的）。

```
沉没成本谬误："我已经在这只股票上亏了 50%，不能卖。"
机会成本思考："如果我现在持有现金，我会买这只股票吗？如果不会，就应该卖出。"
```

## 决策仪表

```
决策时问自己：

- [ ] 我放弃的第二佳选择是什么？
- [ ] 它的预期回报是多少？
- [ ] 我当前的决策，一年后看还值得吗？
- [ ] 如果我有无限现金但时间有限，我会做什么？
```
