# Personal Capital OS

## 这是什么？

Personal Capital OS 是一套**个人资本操作系统**——它不是一本书，而是一个持续演进的知识产品。

它把个人财富、商业认知、决策能力和创造力，系统性地组织成一个可迭代、可升级的思维框架。

## 核心理念

> **聊天 = 研发（Research）**
> **Markdown = 发布（Release）**

每次探索都产生新的认知，每次沉淀都形成稳定版本。几年后回头，看到的不是零散的笔记，而是一套完整记录思维操作系统如何诞生和成长的演化史。

## 四大操作系统

| 系统 | 核心命题 | 关键能力 |
|------|----------|----------|
| **Decision-OS** | 如何做出高质量决策？ | 决策框架、Checklist、避免认知偏误 |
| **Financial-OS** | 如何让财富持续增长？ | 资产配置、企业分析、估值、周期判断 |
| **Business-OS** | 如何理解和建立好生意？ | 商业模式、竞争壁垒、增长引擎 |
| **Creator-OS** | 如何创造长期价值？ | 知识输出、品牌建设、产品思维 |

## 四个底层模型

所有 OS 的运行都建立在四个底层模型之上：

1. **财富模型** — 什么是财富？财富的来源与形态
2. **企业模型** — 什么是好企业？企业的价值创造机制
3. **估值模型** — 什么是合理价格？价格与价值的关系
4. **周期模型** — 什么是周期？宏观、行业、心理周期的叠加

## 版本哲学

- **Stable**：经过充分讨论和验证的稳定内容
- **Draft**：正在撰写中的内容
- **Research**：还在探索阶段的想法

每个版本都有完整的 CHANGELOG，记录新增、修改和删除的内容及原因。
