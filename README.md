# Project Atlas / Personal Capital OS

## 这是什么？

> **Atlas（阿特拉斯）**——在神话中承担整个天空，现代常用来表示地图集和知识体系。

Project Atlas 是一张理解**世界、商业、资本和人生**的地图。它不是一本书，而是一个持续升级的个人操作系统。

Personal Capital OS 是 Project Atlas 的核心子系统，专注于资本、财富、投资与商业。

## 核心理念

> **聊天 = 研发（Research）**
> **Markdown = 发布（Release）**

每次探索都产生新的认知，每次沉淀都形成稳定版本。几年后回头，看到的不是零散的笔记，而是一套完整记录思维操作系统如何诞生和成长的演化史。

## 五大操作系统

| 系统 | 核心命题 | 关键能力 |
|------|----------|----------|
| **Decision-OS** | 如何做出高质量决策？ | 决策框架、Checklist、偏误防护 |
| **Financial-OS** | 如何让财富持续增长？ | 资产配置、企业分析、估值、周期 |
| **Business-OS** | 如何理解和建立好生意？ | 商业模式、护城河、创新增长 |
| **Creator-OS** | 如何创造长期价值？ | 知识输出、品牌建设、产品思维 |
| **Life-OS** | 如何整合人生系统？ | 仪表盘、十层架构、知识图谱 |

## 思维模型库

Models/ 目录下包含 9 个独立可复用的思维模型：

1. **财富模型** — 什么是财富？财富的来源与形态
2. **企业模型** — 什么是好企业？价值创造机制
3. **估值模型** — 什么是合理价格？价格与价值的关系
4. **周期模型** — 宏观、市场、心理三层周期叠加
5. **网络效应** — 为什么用户越多价值越大？
6. **飞轮模型** — 自我加速的增长引擎如何构建？
7. **S-Curve** — 创新的生命周期和第二曲线
8. **反馈循环** — 正反馈是引擎，负反馈是刹车
9. **机会成本** — 任何选择的真正代价

## 版本哲学

- **Stable**：经过充分讨论和验证的稳定内容
- **Draft**：正在撰写中的内容
- **Research**：还在探索阶段的想法

每个版本都有完整的 CHANGELOG，记录新增、修改和删除的内容及原因。
