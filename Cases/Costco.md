# 案例研究：Costco

## 为什么研究 Costco？

Costco 是理解"会员制模式 + 极致运营效率 + 飞轮效应"的经典案例。它证明了"赚最少的商品利润，但赚最多的客户忠诚度"的商业模式。

## 商业模式

- **付费会员**：约 $60-120/年/人，续费率 > 90%
- **极低商品加价**：毛利率上限 14%（强制设定），远低于超市（25-30%）
- **利润结构**：大部分利润来自会员费，商品销售利润覆盖运营成本

## 护城河分析

1. **会员锁定**：续费率 > 90% = 锁定复利
2. **价格优势**：规模效应 → 议价力最大化 → 价格最低
3. **品牌信任**："Costco 的价格一定是最低的"
4. **精选 SKU**（仅 3700 个 vs 沃尔玛 120000 个）= 高效的库存周转
5. **Kirkland 自有品牌**：占收入 30%+，更高的利润且差异化

## 飞轮效应

```
会员费收入
    │
    ▼
更强议价力 → 更低采购价格
    │
    ▼
更低零售价 → 更多会员加入
    │
    ▼
更大采购量 → 更高 SKU 周转
    │
    ▼
（回到：更强议价力）
```

## 投资启示

1. **最好的商业模式是"先付费后服务"**——会员制带来了现金流和预付款
2. **极端聚焦比面面俱到更有效**——精选 SKU = 库存效率 × 议价力
3. **续费率 = 护城河的量化指标**——>90% = 几乎无法攻破
4. **利润率低不代表不赚钱**——Costco 的 ROE 远高于传统零售商

---

*Costco 证明了：让客户觉得赚了，才是最好的商业模式。*
