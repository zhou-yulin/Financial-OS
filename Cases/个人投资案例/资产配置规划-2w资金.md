## 第三步：资产配置方案（基于 Financial-OS 框架）

### 3.1 核心原则（参考 Volume01/01-长期主义 + Volume01/02-复利思维）

```
1. 长期主义：投资期限 10 年，不做短期择时
2. 复利最大化：不中断的复利 > 超高的短期回报
3. 分散化：不依赖任何单一资产
4. 安全边际：以低于内在价值的价格买入
5. 周期意识：当前市场估值处于中等偏下位置
```

### 3.2 推荐配置方案

| 资产类别 | 比例 | 金额(美元) | 推荐标的 | 理由 |
|----------|------|-----------|----------|------|
| 全球股票宽基指数 | 55% | $11,000 | VTI（美股全市场）或 VT（全球） | 核心仓位，享受市场整体增长 |
| 行业/主题ETF | 15% | $3,000 | QQQ（纳斯达克100）或 VGT（科技） | 战术仓位，AI/科技长期趋势 |
| 个股精选 | 10% | $2,000 | 1-2 只你深度研究的好公司 | 卫星仓位，追求超额收益 |
| 债券/固收 | 10% | $2,000 | BND（全债市）或 SHY（短期国债） | 缓冲波动，危机时转换的弹药 |
| 现金/货币基金 | 10% | $2,000 | 高收益储蓄或货币基金 | 保持灵活性，等待市场恐慌加仓 |

### 3.3 配置逻辑详解

**为什么 55% 核心指数？（参考 Volume02/07-资产配置）**

- 研究表明：资产配置解释了超过 90% 的投资收益波动
- 宽基指数是"不赌个股，赌整体经济增长"
- 不需要判断谁赢——只要经济体长期向上，指数就涨
- 美国 200 年历史：股票年化回报约 7-9%

**为什么留 10% 现金？（参考 Models/周期模型）**

- 现金 = 危机中的期权价值
- 2008 年、2020 年、2022 年——每次大跌都是十年一遇的买入机会
- 持有现金不是为了"赚取收益"，而是为了**等待他人恐慌**
- 参考巴菲特：伯克希尔始终持有 3000 亿+现金储备

**为什么 10% 债券？（参考 Models/反馈循环）**

- 债券和股票通常是负相关
- 市场崩盘时，债券（尤其是国债）往往上涨
- 可转换为"危机弹药"——当股市大跌时卖掉债券加仓股票

**为什么 15% QQQ/科技？（参考 Cases/AI.md + Cases/Nvidia.md）**

- AI 处于 S-Curve 的早期成长期（参考 Models/S-Curve）
- 科技板块长期回报率高于市场平均，但波动更大
- 这是一个"风险更高、回报更高"的战术配置

```

3. 分批执行以上买入计划
4. 设定自动再投资（股息自动再投资）
```

## 第五步：持有与再平衡（参考 Financial-OS v1.2）

### 5.1 再平衡规则

- **年度再平衡**：每年 1 月初，将配置调回目标比例
- **阈值触发**：任何类别偏离目标 ±5% → 再平衡
- **下跌响应**：如果整体组合下跌 20%+ → 审视是否需要加仓

### 5.2 卖出规则（参考 Financial-OS）

```
以下情况下卖出：
1. 买入逻辑不再成立（估值严重泡沫、基本面恶化）
2. 明显更好的机会出现（机会成本）
3. 需要再平衡
4. 绝对不要因为涨了而卖出
5. 绝对不要因为跌了而卖出
```

### 5.3 定投机制

```
建议：每个月额外存 $100-500 用于加仓
方式：自动定投 VT 或 VTI（每月固定日期买入）
效果：价格低时自动多买，价格高时自动少买
```

## 每项配置的详细逻辑（调用本书内容）

### 为什么 30% 沪深 300？

**参考：Volume02/07-资产配置 + Financial-OS**

- 沪深 300 是中国最大的 300 家上市公司，代表 A 股的"核心资产"
- 当前 PE 约 12 倍，处于历史低位
- 2005 年以来的年化回报约 8-10%
- 不需要选股，只需要相信中国经济长期向上

**风险提示**：A 股波动大，沪深 300 单年最大回撤曾到 -40%。但拉长到 10 年，正收益概率 > 85%。

### 为什么 15% 创业板/科创50？

**参考：Models/S-Curve + Cases/AI.md + Cases/Nvidia.md**

- 创业板/科创板代表中国的"新经济"——新能源、半导体、AI、生物医药
- 中国在很多科技领域（新能源、光伏、储能）处于全球领先
- 这些行业的 S-Curve 处于成长期，未来 5-10 年仍有空间
- 风险更高（波动率约 30-40%），所以用 15% 配置控制仓位

### 为什么 15% 行业ETF？

**参考：Business-OS + Models/竞争模型**

- 中国有全球最强的两个行业优势：
  - **消费（白酒/食品饮料）**：茅台、五粮液等——护城河来自品牌和文化嵌入
  - **制造（新能源/光伏/新能源车）**：宁德时代、比亚迪等——护城河来自垂直整合和成本优势
- 行业ETF比个股分散风险，比宽基指数更集中

### 为什么 10% 个股？

**参考：Templates/企业分析模板 + Volume02/08-企业分析框架**

- 作为学习研究用——真正学会分析一家企业
- 建议从你了解的行业开始
- 必须通过基本面筛选（参考企业分析模板）

**快速初筛要求**（参考 Financial-OS）：
```
- [ ] 能一句话说清楚这家公司怎么赚钱
- [ ] 有护城河（品牌/转换成本/网络效应/成本优势）
- [ ] 过去 5 年收入持续增长
- [ ] 有息负债率 < 50%
- [ ] ROE > 15%（连续 3 年）
```

### 为什么 15% 债券 + 15% 现金？

**参考：Volume02/11-风险管理 + Models/周期模型 + Volume02/10-周期与择时**

- A 股波动率是美股的 1.5-2 倍 —— **需要更多的安全缓冲**
- 债券在 A 股暴跌时（如 2015 年股灾、2022 年下跌）提供对冲
- 现金 = 危机期权价值
- 2024 年 A 股低迷时，持有现金的人可以在底部加仓——**这就是复利的来源**

## A 股特有风险提示（参考 Volume02/11-风险管理）

```
⚠️ 政策风险：A 股受政策影响大（监管变化、行业整顿）
⚠️ 波动风险：A 股单年可跌 30-40%，心理承受力需要训练
⚠️ 流动性风险：小盘股流动性差，我们只用 ETF 避免了
⚠️ 汇率风险：人民币兑美元波动会影响以人民币计价的收益

应对策略：
- 用宽基 ETF 避免个股踩雷
- 用 15% 现金 + 15% 债券控制整体波动
- 坚持定投，不择时
- 年化预期下调至 6-8%（vs 美股的 7-9%）
```
