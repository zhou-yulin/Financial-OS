# 案例研究：腾讯

## 为什么研究腾讯？

腾讯是理解"社交网络效应 + 生态投资 + 超级应用"的典型案例。微信是世界上最成功的超级应用。

## 商业模式

| 业务 | 核心变现 | 特征 |
|------|----------|------|
| 社交（微信/QQ） | 广告 + 增值服务 | 拥有13亿+月活 |
| 游戏 | 虚拟道具/充值 | 全球最大游戏公司 |
| 金融科技（微信支付） | 支付手续费 + 理财 | 全国第二支付网络 |
| 云/企业服务 | SaaS 订阅 | 第二曲线，快速增长 |
| 投资组合 | 投资收益 | 持有美团/拼多多/京东等 |

## 护城河分析

1. **网络效应（最强）**：微信 = 社交关系网络。你的朋友都在，你不得不在
2. **超级应用生态**：社交 + 支付 + 小程序 + 视频号 = 一人一应用
3. **游戏 IP 矩阵**：Riot（英雄联盟）+ Supercell + 自有研发
4. **投资引擎**：通过资本实现生态护城河

## 关键风险

- 国内监管风险（游戏、金融科技限制）
- 微信增长天花板（渗透率几乎 100%）
- 海外扩张受限
- 投资收益波动大

## 投资启示

**最强的护城河是"你的朋友都在这里"**——微信证明了社交网络的不可替代性。但挑战在于：如何在成熟业务上开启第二增长曲线？
