# 案例研究：Tesla

## 为什么研究 Tesla？

Tesla 是理解"颠覆式创新 + 垂直整合 + 数据网络效应"的最佳案例。它从根本上改变了百年历史的汽车产业。

## 商业模式

- **硬件销售（汽车）**：约 85% 收入，但毛利率受到价格战影响
- **软件/服务（FSD、超充网络）**：高利润、经常性收入潜力
- **能源业务**：太阳能 + 储能（Megapack），第二增长曲线

## 护城河分析

1. **先发优势 + 品牌**：电动车 = Tesla 的代名词
2. **数据网络效应**：数百万车辆在路上采集驾驶数据 → FSD 更准确 → 更多销售
3. **垂直整合**：自研芯片、电池、软件、制造——传统车企无法复制
4. **超充网络**：最大的独家充电网络——物理护城河
5. **成本优势**：一体化压铸 + 4680 电池，制造成本持续下降

## 关键风险

- 估值赋予完美执行预期
- 马斯克个人品牌风险
- 竞争（比亚迪、传统车企转型）
- 价格战压缩毛利率
- 自动驾驶监管不确定性

## 投资启示

**颠覆者不仅要证明自己是对的，还要证明整个行业是错的。** Tesla 已证明前半句，正在证明后半句。
