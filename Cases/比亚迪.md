# 案例研究：比亚迪

## 为什么研究比亚迪？

比亚迪是理解"垂直整合成本优势 + 技术路线选择 + 中国制造全球化"的典型案例。它在新能源车时代，用完全不同的路径挑战了 Tesla 的领先地位。

## 商业模式

- **汽车**：比亚迪秦/宋/汉/唐 + 仰望/腾势（高端线）
- **电池**：刀片电池——自研、自产、自用 + 外供
- **半导体**：自研车规级芯片（IGBT、SiC）
- **手机代工**：为华为/小米等代工（稳定的现金流）

关键差异：比亚迪比任何竞争对手都更**垂直整合**——除了轮胎和玻璃，几乎所有部件都自制。

## 护城河分析

1. **成本垂直整合（最强）**：从锂矿到电池到电芯到芯片到整车，几乎全部自研自产——成本控制力远超竞品
2. **技术路线选择**：DM-i 超级混动 + 刀片电池 + e平台 3.0
3. **规模效应**：2022-2026 年销量从 100 万冲向 500 万——规模力压群雄
4. **中国市场深度**：在 10-30 万主流市场建立了品牌信任

## 关键风险

- 海外市场地缘政治风险
- 品牌溢价天花板——高端化仍在探索
- 过度垂直整合的灵活性挑战
- 国内竞争白热化（小米、华为入局）

## 投资启示

**垂直整合在规模足够大时是最深的护城河**。比亚迪证明了在价格敏感的主流市场中，成本优势可以击败品牌溢价。
