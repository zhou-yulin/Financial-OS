# 案例研究：Netflix

## 为什么研究 Netflix？

Netflix 是理解"订阅模式 + 内容飞轮 + 从分销到原创的转型"的经典案例。它从 DVD 邮寄公司转变为全球最大的流媒体平台。

## 商业模式

- **订阅收入**：约 $15-23/月，全球 2.5亿+ 付费用户
- **广告层（2023+）**：低价订阅 + 广告的新增长曲线
- **内容飞轮**：订阅收入 → 投资内容 → 更多/更好内容 → 更多用户

## 护城河分析

1. **数据驱动的内容投资**：Netflix 知道用户想看什么，比传统制片厂更准确
2. **内容库规模**：全球最大的原创内容库之一
3. **品牌=类型**："Netflix and chill" = 流媒体成为动词
4. **全球发行网络**：覆盖 190+ 国家，实现内容成本分摊

| 时期 | 战略 | 护城河 |
|------|------|--------|
| DVD → 流媒体 | 颠覆式创新 | 先发优势 + 品牌 |
| 授权 → 原创 | 垂直整合 | 独家内容 + 数据飞轮 |
| 无广告 → 广告层 | 第二曲线 | 增加ARPU，覆盖新客群 |

## 关键风险

- 流媒体竞争饱和（Disney+、Apple TV、Max 等）
- 内容成本上升，但提价空间有限
- 用户增长见顶（成熟市场渗透率高）
- 热门剧集依赖创意——难以保证持续产出

## 投资启示

订阅模式的甜蜜点在于：**你只需要每年让用户觉得"值得"1-2 次**。Netflix 依靠优质内容维持着极低的退订率。
