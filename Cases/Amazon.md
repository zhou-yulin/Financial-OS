# 案例研究：Amazon

## 为什么研究 Amazon？

Amazon 是理解"飞轮效应 + 第二曲线 + 长期主义"的最佳案例。它从在线书店演变为全球最大的电商平台，同时开创了云计算产业。

## 商业模式

| 业务 | 收入占比 | 利润率 | 特征 |
|------|----------|--------|------|
| 电商（自营+第三方） | 约 60% | 低（<5%） | 飞轮核心，GMV 5000亿+ |
| AWS 云计算 | 约 20% | 高（30%+） | 真正的利润引擎 |
| 广告 | 约 10% | 极高 | 增长最快的业务 |
| Prime/订阅 | 约 10% | 中高 | 提升粘性和LTV |

## 三大飞轮

### 电商飞轮
更低价格 → 更多客户 → 更多卖家 → 更好选择 → 更好体验 → 更多流量 → 更低成本结构 →（循环）

### Prime 飞轮
会员费 → 免费配送+视频+音乐 → 更高LTV → 会员在Amazon消费更多 → 更多商品加入Prime → 更多会员

### AWS 飞轮
更多客户 → 更多使用 → 更多数据/反馈 → 更多服务 → 更优性价比 →（循环）

## 护城河分析

1. **规模效应**：全球最大的仓储物流网络，无人能及
2. **AWS 生态锁定**：开发者在AWS上的投入形成转换成本
3. **Prime 粘性**：2亿+ 会员，续费率>90%
4. **数据优势**：消费者行为数据量全球第一
5. **飞轮互联**：三个飞轮各自加速且相互增强

## 关键成功因素

- **贝索斯的长期主义**：连续20年亏损/低利润投资未来
- **从零售到科技的转型**：AWS 证明了第二曲线的力量
- **客户执念**："客户永远不满足" 推动了所有创新

## 投资启示

1. **飞轮需要耐心**：Amazon 20 年微利，然后爆发
2. **第二曲线来自核心能力**：AWS 用的是Amazon自己的基础设施经验
3. **护城河可以叠加**：电商飞轮 + Prime 粘性 + AWS 锁定 = 无法复制
4. **长期主义的复利**：2000 年投资 Amazon，回报超过 2000 倍
