# 第一卷：认知基础

## 导言

在进入任何具体的操作系统之前，我们需要先建立认知的基础。

如果思维是一栋大楼，这一卷就是地基。地基不牢，楼越高越危险。

## 为什么需要认知基础？

大多数人对财富、商业、决策的理解，停留在"术"的层面——学估值公式、背商业模式画布、记决策框架。

但真正拉开人与人差距的，从来不是"术"，而是"道"。

"道"是什么？

1. **时间观** — 你如何看待时间的价值
2. **长期主义** — 为什么长期比短期重要
3. **复利思维** — 理解世界上最重要的力量
4. **概率思维** — 接受不确定性并与之共舞
5. **第一性原理** — 穿透表象看本质
