# 第四章：第一性原理

## 前言

大多数人思考问题时用的是**类比思维**：这件事像什么？别人是怎么做的？历史上类似的情况是什么？

类比思维有用，但它有一个致命缺陷：**你永远在被已有的框架限制。**

第一性原理是反过来的：**把问题拆解到最基础的真理，然后从那里重新构建。**

## 一、什么是第一性原理？

### 1.1 第一性原理的定义

第一性原理（First Principles Thinking）源自物理学：**把事物分解到最基本的不可再分的基本事实，然后从这些基本事实出发推导结论。**

亚里士多德说：「第一性原理是事物的基础前提，不能被省略，不能被违背。」

### 1.2 第一性原理 vs 类比思维

| | 类比思维 | 第一性原理 |
|------|----------|------------|
| 出发点 | "这件事像什么？" | "这件事的基本事实是什么？" |
| 方法 | 参考已有方案 | 从零开始构建 |
| 优势 | 快速、节省能量 | 创新、突破框架 |
| 劣势 | 路径依赖、渐进式 | 认知消耗大、慢 |
| 适合场景 | 日常决策 | 战略决策、创新 |

类比思维让你成为一个好的模仿者。第一性原理让你成为一个真正的思想者。

### 1.3 埃隆·马斯克的例子

马斯克是当代最著名的第一性原理实践者。

当他开始做 SpaceX 时，所有人都告诉他火箭太贵了。传统思路（类比思维）是：火箭就是这么贵，看看 NASA 和波音就知道了。

马斯克的第一性原理分析：

- 火箭由什么构成？铝合金、钛、铜、碳纤维
- 这些原材料成本是多少？只占火箭价格的 2%
- 为什么成品这么贵？因为供应链冗余、体制成本
- 如果我自己造呢？SpaceX 把火箭成本降低了 90%

**类比的结论是"不可能"，第一性原理的结论是"可以，但需要换一种方式"。**

## 二、如何运用第一性原理

### 2.1 第一步：质疑假设

每个问题背后都有一堆默认假设。第一性原理的第一步是**识别这些假设并质疑它们**。

"投资需要分散化" → 为什么？这个假设从哪来的？
"学投资必须懂财报" → 真的吗？有没有不懂财报但投资很成功的人？
"人应该买房" → 这是一个物理定律，还是一个文化习惯？

每一个被普遍接受的观点，都值得被质疑一次。

### 2.2 第二步：分解到基本事实

把复杂问题拆解到最基础的组成部分。

- "什么是好企业？" → 分解为：什么是收入？什么是成本？什么是竞争？什么是客户？
- "什么是财富？" → 分解为：什么是资源？什么是交换？什么是价值？
- "什么是风险？" → 分解为：什么是永久损失？什么是波动？什么是未知？

你能分解到多深，取决于你对基本事实的理解有多深。

### 2.3 第三步：从基本事实重新构建

知道了基本事实之后，**忘记所有现成的解决方案**，从零开始构建。

这就是最难的部分。因为大脑会不停地往回拉："但别人都是这么做的"，"这不符合常规想法"。

第一性原理需要**勇气**：敢于得出与众不同的结论。

### 2.4 第四步：验证和迭代

第一性原理不是让你无视现实。推导出的结论需要在现实世界中验证。

如果现实与你的推导不符，要么是你的基本事实有误，要么是你的推导有误。回头检查，重新来过。

## 三、第一性原理在投资中的应用

### 3.1 挑战估值公式

所有人都在用 PE、PB、PS 估值。第一性原理的问题是：

- 为什么 PE 是合理的估值工具？
- PE 的基本假设是什么？（盈利可持续）
- 如果这个假设不成立呢？（周期性公司、成长型公司、亏损公司）

当你用第一性原理审视估值，你会发现：**没有一种估值方法适用于所有公司。你需要为每一种企业定制估值框架。**

### 3.2 挑战"买入并持有"

"买入好公司并永远持有"——这是巴菲特带来的智慧。但第一性原理的问题是：

- "永远"是什么意思？物理上可能吗？
- 公司会永远好下去吗？历史上的伟大公司有哪些还在？
- 这个策略的适用条件是什么？

结论不是推翻"买入并持有"，而是**明确它的适用边界**：只有在企业护城河持续扩大的前提下，才成立。

### 3.3 重新定义"风险"

传统金融学定义风险为波动率（Volatility）。但第一性原理告诉你：

- 波动不等于风险。价格下跌 50% 但不永久损失 ≠ 风险
- 真正的风险是：永久性资本损失
- 永久性资本损失的来源：商业模式被破坏、过度杠杆、欺诈

**用第一性原理重新定义风险，你会发现很多"高风险"的资产实际上是低风险的，反之亦然。**

## 四、第一性原理的局限性

### 4.1 不适用于所有场景

日常决策使用第一性原理是过度浪费。你不需要为"今天吃什么"做第一性原理分析。

**第一性原理是战略工具，不是日常工具。**

### 4.2 需要深厚的知识储备

如果你连基本事实是什么都不知道，第一性原理就无从谈起。

这也是为什么第一性原理的使用者往往是某个领域的深度专家。

### 4.3 孤独

第一性原理得出的结论往往与社会共识相反。坚持这些结论需要承受相当大的孤独感和压力。

但这也是超额收益的来源：**共识已经定价，分歧才产生机会。**
