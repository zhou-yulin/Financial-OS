# 第三章：概率思维

## 前言

世界不是确定性的。

但我们的教育体系、思维习惯、甚至语言，都建立在确定性假设上：A 导致 B，好决定带来好结果，努力工作就会成功。

**现实是：好决定可能带来坏结果，坏决定可能带来好结果。**

概率思维就是接受这一点，并学会在不确定性中做决策。

## 一、确定性幻觉

### 1.1 为什么我们渴望确定性？

进化心理学给出了答案：我们的祖先需要确定性来生存。

"那片草丛里是不是有蛇？"——如果你不确定就往前走，可能就死了。所以大脑进化出了"宁可误报，不可漏报"的机制。

**确定性 = 生存。**

但我们所处的现代世界，充满了概率事件。投资、创业、职业选择——没有任何事情是 100% 确定的。而我们的石器时代大脑，仍然在用确定性的方式思考。

### 1.2 结果偏差

这是概率思维最大的敌人。

**结果偏差（Outcome Bias）**：根据结果来反推决策质量。

- 一个人买了彩票中奖了 → "他的决策是对的"
- 一个人投资了一家好公司但亏了 → "他做错了"

这是完全错误的。

**正确的问题不是「结果是什么」，而是「决策过程是否正确」。**

如果决策过程是正确的，即使结果不好，也不应该后悔。你只是遇到了小概率的坏运气。

### 1.3 另一个极端：概率虚无主义

有些人走向了另一个极端：既然一切都是概率，那做什么都没有意义，反正都是运气。

这是对概率思维的误解。

概率的含义不是"一切随机"，而是**你可以通过正确的决策过程，持续提高胜率**。

如果你每次的胜率是 60%，做一百次决策之后，你的总体收益是正的——即使中间可能连续输十次。而那些胜率只有 40% 的人，短期可能赢，长期必然输。

**概率思维不是放弃努力，而是把努力放在提高胜率上。**

## 二、概率决策框架

### 2.1 贝叶斯更新

贝叶斯公式：P(A|B) = P(B|A) × P(A) / P(B)

通俗地说：**新信息出现时，更新你的信念。**

- 你相信某公司是好公司（先验概率）
- 它连续五年业绩增长（新证据）
- 你更新你的信念：这家公司确实是好公司（后验概率）

贝叶斯更新的关键是：**保持开放，随时根据新信息调整判断。**

这与固执己见是相反的。固执己见是"我不管新信息是什么，我的判断不变"。

### 2.2 期望值思维

**期望值 = 概率 × 收益 / 损失**

这是一个强大的决策工具：

- 概率低但收益巨大的机会（如早期投资）
- 概率高但收益适中的机会（如指数定投）
- 概率不确定但损失有限的机会（如学习新技能）

关键不是追求每次都对，而是追求**期望值为正的决策组合**。

你不需要每次下注都赢，你只需要：赢的时候赢得多，输的时候输得少。

### 2.3 反脆弱

纳西姆·塔勒布提出的概念：**反脆弱不是"坚固"，而是"从波动中获益"**。

- 坚固的系统：不受波动影响（如钢筋混凝土）
- 脆弱的系统：被波动摧毁（如过度杠杆的组合）
- 反脆弱的系统：波动越大，收益越大（如拥有大量现金等待市场崩盘）

概率思维的最高境界不是预测，而是**构建一个从不确定性中获益的系统**。

## 三、投资中的概率思维

### 3.1 安全边际

格雷厄姆说："安全边际就是用远远低于内在价值的价格买入。"

换算成概率语言：**安全边际 = 给自己的判断错误留余地。**

你认为一家公司值 100 元，但你在 60 元买入。这意味着即使你的估值错了 40%，你仍然不会亏钱。

安全边际承认了一个事实：**你可能是错的。**

### 3.2 仓位管理

凯利公式：f* = (bp - q) / b

- b：赔率（赢的时候赚多少）
- p：胜率
- q：1 - p（输的概率）
- f*：最优下注比例

核心结论：

- 胜率高、赔率大的机会 → 重仓
- 胜率高、赔率小的机会 → 适度配置
- 胜率低、赔率大的机会 → 小额投机
- 胜率低、赔率小的机会 → 不参与

**永远不要在一件事上押上全部。无论胜率多高，总有黑天鹅。**

### 3.3 穿越周期

短期是投票机，长期是称重机。

短期内，市场受情绪、流动性、叙事驱动——近乎随机。长期内，价格会向价值回归。

概率思维告诉你：**不要预测短期，但可以押注长期均值回归。**
