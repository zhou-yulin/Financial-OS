# 第十三章：商业模式设计

## 前言

这世界上有两种人：一种是被商业模式困住的人，一种是设计商业模式的人。

理解商业模式，不只是为了投资。更是为了你理解**钱在商业世界中究竟如何流动**。

## 一、商业模式的本质

### 1.1 商业模式 = 如何可持续地赚钱

这是一个比你想象的更深刻的问题。

每家公司都有产品或服务，但只有少数公司知道如何**系统性地、重复地、可持续地**变现。

商业模式回答五个问题：
1. 你的客户是谁？
2. 你提供什么价值？
3. 你如何触达客户？
4. 你如何盈利？
5. 你如何保持盈利？

### 1.2 好的商业模式的特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 经常性收入 | 客户不是一次性消费 | SaaS 订阅、会员费、版税 |
| 低成本扩张 | 规模增长不需要相应增加成本 | 软件产品、数字内容 |
| 高客户粘性 | 客户不容易离开 | 微信、Office 365 |
| 正向现金流 | 先收款后服务 | 保险公司、订阅服务 |
| 网络效应 | 使用者越多价值越大 | 社交网络、市场平台 |

一个拥有多个特征的商业模式 = 一台强大的复利机器。

## 二、商业模式类型

理解不同商业模式在收入和可持续性上的根本差异。

### 2.1 一次性销售

客户买一次，很少再买。这需要不断寻找新客户。

- 优势：单笔收入可能较高
- 劣势：营销成本高，增长依赖不断获取新客

➡ 适合高客单价产品（汽车、房产）。

### 2.2 订阅/经常性收入

客户持续付费获得持续服务或产品。

- 优势：可预测收入、累积增长的客户基础
- 劣势：需要持续提供价值，客户流失需要管理

➡ 适合软件（SaaS）、内容（Netflix/Spotify）、会员（Amazon Prime）。

### 2.3 平台/市场

匹配买家和卖家，从交易中抽取佣金。

- 优势：轻资产、可规模化、网络效应
- 劣势：需要同时吸引两方用户（鸡和蛋的问题）

➡ 适合交易型场景（电商平台、支付系统、共享经济）。

### 2.4 交易/经纪

从每笔交易中抽取百分比。

- 优势：收入与交易规模绑定，随市场增长而增长
- 劣势：受交易量波动影响大

➡ 适合金融市场、房产中介、投资银行。

### 2.5 制造/实物产品

生产实物产品并销售。

- 优势：可建立品牌、知识产权保护
- 劣势：扩张需要大量资本、库存风险

➡ 适合有品牌护城河的消费品。

## 三、设计你自己的商业模式

### 3.1 商业模式画布

一个简化版的分析框架：

| 层面 | 你需要回答的问题 |
|------|------------------|
| 价值主张 | 你要解决什么问题？为谁解决？ |
| 客户细分 | 谁是最愿意付费的客户？ |
| 客户关系 | 客户如何发现你？你如何保持关系？ |
| 收入流 | 客户因为什么付钱？频率如何？ |
| 关键资源 | 你需要什么才能交付价值？ |
| 关键活动 | 你每天在做什么来保持运行？ |
| 成本结构 | 哪些成本是一次性的，哪些是重复的？ |
| 合作伙伴 | 谁能帮你扩大规模？ |

### 3.2 从技能到商业

如果你想从零开始建立自己的商业模式：

1. **识别你的核心技能和知识**
2. **找到愿意为这些技能付费的人**（市场规模）
3. **设计交付方式**（可规模化的一对多？还是一对一？）
4. **测试定价**（客户愿意付多少？频率？）
5. **自动化收入和交付**（减少依赖你的个人时间）

最好的商业模式是：你只需做一次产品/内容，就能持续销售。
