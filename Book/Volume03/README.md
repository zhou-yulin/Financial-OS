# 第三卷：商业与创造

## 导言

第一卷建立了认知基础。第二卷将认知基础应用于投资体系。

第三卷是认知基础向最后的、也是最重要的领域的延伸：**商业和创造**。

## 为什么商业与创造是一体的？

商业 = 识别和利用价值创造的机会。
创造 = 从零开始生成有价值的东西。

两者本质上是同一枚硬币的两面。一个企业家必须创造才能建立商业。一个内容创造者必须理解商业才能将创造力变现。

**商业是创造的放大器。创造是商业的源头。**
