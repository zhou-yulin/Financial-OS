# 第十五章：创新与增长

## 前言

企业有两种增长方式：在现有轨道上做得更好（优化），或者开辟新的轨道（创新）。

优化让你活下去。创新让你跳到下一个量级。

## 一、创新的类型

### 1.1 渐进式创新

在已有产品或服务上做微小但持续的改进。

- iPhone 从 12 到 13 到 14 到 15
- 汽车燃油效率每年提升 1%
- 银行 APP 增加新功能

渐进式创新的价值在于**保持竞争优势**，而不是创造新的优势。它让你不掉队。

### 1.2 颠覆式创新

创造全新的产品或服务方式，从根本上改变了行业规则。

- Netflix：从 DVD 邮寄到流媒体（颠覆了整个影视发行）
- iPhone：将电话变成个人计算平台（颠覆了多个行业）
- 特斯拉：将汽车重新定义为"带轮子的手机"（颠覆了汽车产业）

颠覆式创新极少见。大多数公司永远不会经历。但理解它让你识别机会和威胁。

### 1.3 商业模式创新

用新的方式赚钱，而不是创造新的产品。

- Dollar Shave Club：订阅剃须刀（新模式，旧产品）
- Spotify：流媒体 = 无限访问而非购买单曲（改变了音乐行业的收入模式）
- Airbnb：让任何人都能成为酒店（资产轻量化）

商业模式创新 = 用新方式回答"客户为什么付钱？"

### 1.4 体验创新

改变客户与产品或服务的交互方式。

- 苹果零售店：将电脑销售变成体验空间
- 星巴克：将咖啡变成"第三空间"
- Disney+：将内容消费变成无缝沉浸体验

体验创新不需要新技术。它需要**对客户心理的深刻理解**。

## 二、增长的来源

### 2.1 内生增长 vs 外延增长

**内生增长**：从已有业务中增长
- 扩大市场份额
- 提价（有定价权的公司）
- 增加客户消费量

**外延增长**：通过收购并购增长
- 买下竞争者
- 进入新市场
- 获得新技术或能力

内生增长更可持续，因为它是靠自身实力。外延增长有整合失败的风险。

### 2.2 增长的极限

每一种增长模式都有天花板。

- 市场饱和：已经没有更多客户可获取
- 竞争加剧：增长带来更多模仿者
- 大公司病：组织臃肿、决策缓慢

**能持续增长的公司，是那些能不断找到新增长引擎的公司。**

### 2.3 第二曲线

查尔斯·汉迪提出的概念：

在第一曲线（当前业务）还在增长时，就开始培育第二曲线（未来业务）。

- Amazon：从在线书店 → 全品类电商 → AWS 云计算
- 苹果：从 Mac → iPod → iPhone → 服务生态
- 微软：从 Windows → Office → Azure 云

**最危险的不是现在的业务不行了，而是现在的业务还在行的时候，你忘了寻找第二曲线。**
