# 第二卷：投资体系

## 导言

第一卷建立了认知基础。第二卷是认知基础在投资领域的应用。

投资不是预测，不是赌博，不是跟风。投资是一套完整的体系。

## 为什么需要投资体系？

没有体系的投资，是情绪驱动的。市场涨了买，市场跌了卖，追涨杀跌，不断交学费。

有体系的投资，是规则驱动的。你不需要知道明天市场怎么走，你只需要知道：当条件 X 满足时，执行策略 Y。

**体系不是保证你每次都赢。体系是保证你长期能赢。**
