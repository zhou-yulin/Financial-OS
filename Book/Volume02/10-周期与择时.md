# 第十章：周期与择时

## 前言

周期是投资中最被低估的概念。

大多数人花 90% 的时间研究"买什么"，花 10% 的时间思考"什么时候买"。

但巴菲特的老师格雷厄姆说：“**买入价格决定了你的回报率。**”

## 一、周期的三层结构

### 1.1 宏观经济周期

经济不会直线增长，而是在繁荣和衰退之间波动。

典型的经济周期四阶段：
1. **复苏**：从衰退中走出，信心恢复，企业投资回升
2. **繁荣**：经济增长加速，就业充分，信贷扩张
3. **过热**：通胀上升，央行加息，资产价格高涨
4. **衰退**：经济增长放缓甚至收缩，失业上升，信贷收紧

经济周期通常持续 5-10 年。但现代央行的干预使得周期变得更难判断。

### 1.2 市场周期

股票市场的周期比经济周期更剧烈、更情绪化。

市场周期的本质是**估值周期**：
- 熊市底部：PE 被极度压缩，通常在 8-12 倍
- 牛市顶部：PE 膨胀到 20-30 倍甚至更高

同一个企业，在两个不同时间点，市场愿意付出的价格可能差 3-5 倍——即使这个企业的价值没有变化。

**市场周期的波动幅度远超企业价值的波动幅度。**

### 1.3 心理周期

这是最深层的周期，也是驱动市场周期的元因素。

投资者心理在两个极端之间摇摆：

| 底部心理 | 顶部心理 |
|----------|----------|
| 恐惧 | 贪婪 |
| "这次不一样，永远不会涨了" | "这次不一样，永远不会跌了" |
| 对好消息视而不见 | 对坏消息视而不见 |
| 现金为王 | 杠杆满仓 |
| 没人讨论股票 | 所有人都讨论股票 |

**心理周期是最可预测的周期，因为它源于永恒不变的人性。**

## 二、周期判断的方法

### 2.1 估值信号

估值是判断周期位置最可靠的指标：

- 整体市场 PE 处于历史什么位置？（高位 1/3 还是低位 1/3？）
- 席勒 CAPE 比率（周期调整市盈率）在什么位置？
- 股债收益率对比（股票收益率与国债收益率比较）

**估值不能告诉你精确的时间，但能告诉你当前的风险收益比。**

### 2.2 行为信号

这些都是市场情绪的"温度计"：

- 身边从不关心投资的人开始问"买什么股票好"？→ 可能过热
- 媒体大量报道"XX 炒股月入 X 万"？→ 极度危险
- 投资群都在哀嚎"血亏"、"再也不碰了"？→ 可能接近底部
- IPO 数量激增，质量下降？→ 市场过热
- 回购潮出现，内部人增持？→ 可能低估

### 2.3 流动性信号

央行货币政策是市场周期的重要驱动力：

- 加息周期：流动性收紧 → 估值下行压力
- 降息周期：流动性放松 → 估值上行空间
- 信用利差扩大：市场避险情绪升温
- 信用利差缩小：风险偏好上升

## 三、择时的真相

### 3.1 为什么大多数人的择时是失败的？

研究一致表明：**个人投资者在择时上亏的钱比选股上亏的更多。**

原因：
- 情绪驱动：涨了追高，跌了恐慌割肉
- 过度交易：越频繁交易，收益越差
- 认知偏误：把运气当能力，把短期当趋势

### 3.2 真正的择时是什么？

真正的择时不是预测明天涨还是跌。真正的择时是**识别市场的极端位置**。

- 当估值进入历史低位 20%，心理极度恐惧 → 这可能是好的买入时机
- 当估值进入历史高位 20%，心理极度狂热 → 这可能是考虑减仓的时机

择时的目标是**不犯大错**，而不是精确地找到顶部和底部。

### 3.3 定投：普通投资者的最优择时策略

如果你承认自己无法判断周期，或者不想花时间判断周期，定投（Dollar Cost Averaging）是最好的解决方案。

**定投 = 在价格低的时候自动多买，在价格高的时候自动少买。**

你不需要判断任何东西。你只需要纪律和耐心。
