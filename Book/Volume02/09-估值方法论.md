# 第九章：估值方法论

## 前言

你找到了一家好企业。但"好企业"不等于"好投资"。

**好企业 + 坏价格 = 坏投资。**
**一般企业 + 好价格 = 可能的好投资。**

估值就是回答一个核心问题：**我愿意为这家企业付多少钱？**

## 一、估值的本质

### 1.1 估值不是数学

很多人以为估值就是拿 Excel 做一个 DCF 模型，算出一个精确的数字，然后低于这个数字就买入。

这是对估值的最大误解。

**估值是一种思维方式，不是一个数学结果。**

巴菲特的合伙人查理·芒格说："我从来没有见过巴菲特用 Excel 做估值。"巴菲特在脑子里完成估值。

估值的目的是建立**安全边际**——为自己可能的错误留出余地，而不是找到"精确的内在价值"。

### 1.2 市值 vs 内在价值

**市值**：市场当前给公司的定价。由情绪、流动性、叙事驱动，变化剧烈。

**内在价值**：公司未来所有现金流的现值。由公司的创造价值能力决定，变化缓慢。

**投资的本质：当市值远低于内在价值时买入，当市值远高于内在价值时卖出。**

格雷厄姆的比喻精妙绝伦："市场是你的仆人，不是你的导师。"你不应该问市场"这个值多少钱"，你应该自己判断值多少钱。

## 二、主要估值方法

### 2.1 现金流折现法（DCF）

**核心逻辑**：公司的价值等于它未来所有现金流的现值。

公式：V = Σ(CF_t / (1+r)^t)

这里的关键挑战：
- 你真的能预测未来 10 年的现金流吗？（大概率不能）
- 折现率选什么？（差 1% 结果差很多）
- 终值假设是什么？（可能占了 80% 的估值结果）

**DCF 的局限**：它给了你一个精确的数字，但这个数字完全依赖于你的假设。精确的错误不比模糊的正确好。

DCF 最适合：现金流稳定、可预测的公司（消费品、公用事业）。最不适合：高增长、不确定性大的公司。

### 2.2 相对估值法（PE/PB/PS）

**核心逻辑**：与同类公司对比，判断是否高估/低估。

常见指标：
- PE（市盈率）：股价 / 每股收益。最常用，但不适用于周期股和亏损公司。
- PB（市净率）：股价 / 每股净资产。适用于金融公司。
- PS（市销率）：股价 / 每股营收。适用于高增长不盈利的公司。
- PEG：PE / 增长率。试图把增长纳入估值。

**相对估值的局限**：如果整个行业都被高估（或低估），你就失去了锚点。2000 年的互联网泡沫就是例子——所有互联网公司用相对估值看都"合理"。

### 2.3 反向 DCF

**核心逻辑**：不从假设推估值，而是**从当前市值反推市场隐含的增长率**。

- 当前市值 1000 亿
- 反推：市场假设这家公司未来 10 年每年增长多少？
- 如果你认为市场假设的增长率过高（或过低），就产生了判断

反向 DCF 是一个强大的工具，因为它把问题从"值多少钱"变成了"市场在预期什么，这个预期合理吗？"

### 2.4 分类估值法

不同类型的企业用不同的方法：

| 企业类型 | 推荐方法 | 原因 |
|----------|----------|------|
| 成熟消费品 | DCF + PE | 现金流稳定，可预测 |
| 成长科技 | PS + 反向DCF | 利润低，市销率更相关 |
| 金融/银行 | PB + ROE | 资产是关键 |
| 周期股 | 平均利润 PE | 不能用顶部或底部利润 |
| 初创/无利润 | 市场规模估算 | 不确定性极大 |

## 三、估值的黄金法则

### 3.1 宁要模糊的正确，不要精确的错误

模糊地知道一家公司大概值 800-1200 亿，比精确地算出一个"1234.56 亿"更有用。

### 3.2 估值范围比估值数字更重要

一个好的估值应该给出一个范围：在悲观情境下值多少？中性？乐观？

如果三种情境下你都觉得便宜，那才是真的便宜。

### 3.3 安全边际是最重要的

无论用什么估值方法，都要保证有足够的安全边际。

- 格雷厄姆：深度价值投资，通常要求 50% 以上的安全边际
- 巴菲特：合理价格买好公司，安全边际来自企业质量
- 彼得·林奇：PEG < 1 是安全边际

**最糟糕的错误不是估值错误，而是在没有安全边际的情况下投资。**

### 3.4 贵和便宜是一个相对问题

在利率为 0 的时代，PE 20 可能合理。在利率为 5% 的时代，PE 20 可能昂贵。

估值永远要考虑**机会成本**：这笔钱的替代用途是什么？
