# 第十一章：风险管理

## 前言

投资的第一个原则是不亏钱。第二个原则是记住第一个。

这不是说你的投资永远不出现账面亏损。而是说：**你永远不应该遭受永久性资本损失。**

## 一、重新定义风险

### 1.1 风险 ≠ 波动

金融教科书告诉你风险是波动率（Volatility）。这是为了让数学模型好看，而不是为了让你赚钱。

一只价格稳定上涨的股票，波动率可能很低，但如果它的商业模式正在被颠覆，它就是高风险的。

一只价格剧烈波动的股票，如果它的竞争优势在持续增强，长期的永久性损失风险反而是低的。

**真正的风险只有一个：永久性资本损失。**

### 1.2 永久性损失的四个来源

1. **商业模式破坏**：新技术或竞争使得公司无法继续盈利
2. **过度估值**：以过高的价格买入，即使公司依旧经营，也需要多年才能回本
3. **过度杠杆**：在市场下跌时被迫卖出（爆仓）
4. **欺诈**：管理层欺骗投资者，财务造假

理解这些来源——你就可以系统性地防范它们。

### 1.3 风险是不可消除的

任何投资都有风险。追求零风险的结果是不投资，而完全不投资也是一种风险——通胀会慢慢侵蚀你的购买力。

**风险管理的目的不是消除风险，而是识别风险、评估风险、并将风险置于可控范围内。**

## 二、风险管理工具箱

### 2.1 安全边际

这是最重要的风险管理工具。

安全边际 = 以远低于内在价值的价格买入。

- 如果你对企业的估值是 100 元，你应该在 60 元甚至 50 元才买入
- 这个 40-50% 的折扣就是对冲你判断错误的风险
- 即使你错了 40%，你仍然不会损失本金

安全边际的宽度取决于你对企业的自信程度。越不确定，要求的安全边际越大。

### 2.2 分散化

不要把鸡蛋放在一个篮子里——这句话简单但深刻。

分散化分三个层面：

- **资产类别分散**：股票 + 债券 + 现金 + 其他
- **地域分散**：不同国家、不同货币
- **个股分散**：单只股票不超过组合的 10-20%

但注意：分散化不是越多越好。持有 500 只股票的风险收益比未必好过持有 15-20 只精选股票。过度分散化 = 你只是在做昂贵的指数基金。

### 2.3 仓位管理

这是职业投资者和业余投资者最大的区别。

仓位应该与你的信心成正比：

- 高确定性 + 大安全边际 = 重仓（但不要超过 20%）
- 中等确定性 + 合理安全边际 = 标准仓位
- 低确定性 + 窄安全边际 = 观察仓位

仓位管理的关键纪律：**永远不要因为已经涨了就追加仓位。永远不要因为你已经亏了就追加仓位来摊平成本。**

### 2.4 止损策略

止损不等于"跌了 20% 就卖"。

真正的止损是**逻辑止损**：当你买入的逻辑不再成立时，无论价格如何，都应该卖出。

- 买入逻辑：公司护城河在扩大。如果发现护城河在缩小 → 止损。
- 买入逻辑：管理层会执行战略。如果管理层失职 → 止损。
- 买入逻辑：估值安全边际充足。如果发现之前的估值假设有重大错误 → 止损。

价格的涨跌不是止损的理由。逻辑的改变才是。

### 2.5 不要用杠杆

杠杆是投资者最大的敌人。

没有杠杆，市场下跌 50% 是一个购买机会。有杠杆，市场下跌 50% 意味着你被强制清仓——即使你是对的，你也无法等到被验证的那一刻。

巴菲特的成功有一个前提：**他从不使用杠杆。**

## 三、黑天鹅的应对

### 3.1 黑天鹅的特征

塔勒布定义的黑天鹅：**罕见、影响力巨大、事后看起来可以解释但事前无法预测的事件。**

2008 年金融危机、2020 年新冠疫情——这些都属于黑天鹅。

### 3.2 应对策略

你无法预测黑天鹅。但你可以为黑天鹅做好准备：

1. **保持充足的现金储备**（至少 6-12 个月生活开销）
2. **不把所有资产放在同一类资产中**
3. **购买"看跌期权"式配置**：5-10% 的黄金或国债，在危机中往往表现较佳
4. **保持灵活**：能够在危机出现时改变策略

### 3.3 反脆弱心态

危机不是惩罚，而是**低价获取优质资产的机会**。

真正的投资者在危机中看到的是打折、是礼物、是十年一遇的买入机会。

这种心态需要在日常就建立。你不能在市场崩盘时才培养反脆弱心态——那时候你已经被恐惧淹没了。
