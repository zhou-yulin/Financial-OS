# Financial-OS：金融操作系统

## 概述

Financial-OS 是你的财富管理中枢。它将投资从一个"偶尔的行为"升级为一个持续运行的系统。

## 核心原则

1. **长期主义**：以十年为基本单位思考，而不是以天
2. **安全边际**：永远为自己的错误留余地
3. **分散化**：不依赖任何单一资产或策略
4. **复利最大化**：不中断的复利 > 超高的短期回报
5. **周期意识**：知道自己在市场周期中的位置

## 资产配置框架

### 目标配置比例

| 资产类别 | 目标比例 | 实现方式 |
|----------|----------|----------|
| 全球股票（宽基指数） | 50-60% | QQQ, SPY, VT 等 |
| 行业/主题配置 | 15-20% | 科技、消费、医疗等板块 |
| 个股精选 | 10-15% | 深度研究后的持仓（≤5只） |
| 债券/固收 | 10-15% | 国债、投资级公司债 |
| 现金/替代 | 5-10% | 现金、黄金、加密货币 |

### 生命阶段配置调整器

根据年龄和财务状况，调整权益类 vs 固收类比例：

| 生命阶段 | 特征 | 权益配比 | 固收配比 | 核心策略 |
|----------|------|----------|----------|----------|
| 积累期（20-35） | 收入上升、支出增长 | 80-90% | 5-10% | 高仓位、低频率、学习为主 |
| 增长期（35-50） | 收入高峰、支出稳定 | 65-80% | 10-20% | 龙头集中、适度杠杆（5%以下） |
| 保护期（50-60） | 考虑退休、保护本金 | 50-65% | 20-30% | 减少风险、增加分红资产 |
| 享用期（60+） | 退休、有固定支出 | 30-50% | 30-50% | 现金流优先、稳定性优先 |

**如何计算你的真实风险承受能力**：

```
计算指标：
- 如果你的投资组合下跌 30%，你的净资产会下降 _____%
- 下跌后的资产仍然可以支持你 _____ 年的生活
- 如果市场三年不反弹，你能不被迫卖出吗？ □ 能 □ 不能
```

### 市场周期调整框架

```
市场位置 → 调整行为

极度恐惧（市场底部信号）
    → 提高权益比例 +10%（增持指数）
    → 拉长持有周期

恐慌（短期下跌）
    → 保持比例，不做恐慌操作
    → 检查买入逻辑是否改变

正常
    → 维持目标比例
    → 每季度再平衡

贪婪（市场过热信号）
    → 降低权益比例 -5%
    → 增加现金储备

极度贪婪（市场顶部信号）
    → 降低权益比例 -10%
    → 提高现金比例，等待机会
```

## 企业筛选标准（快速初筛）

在深入分析之前，先用五个问题过滤：

1. **我能用一句话解释这家公司怎么赚钱吗？**
2. **这家公司有护城河吗？是什么？**
3. **它的收入在过去 5 年是增长的吗？**
4. **它的负债合理吗？（有息负债/总资产 < 50%）**
5. **管理层有持股吗？他们诚实吗？**

如果任何一条回答"否"或"不确定" → 不做深入分析。

## 卖出标准

以下情况下应该考虑卖出：

1. **买入逻辑不再成立**（护城河缩小、管理层变更等）
2. **找到明显更好的机会**（机会成本）
3. **估值严重偏离内在价值**（过度高估）
4. **需要再平衡**
5. **绝对不要因为涨了而卖出，也不因为跌了而卖出**

## 风险管理清单

```
- [ ] 单只股票仓位 < 20%
- [ ] 行业集中度 < 40%
- [ ] 不使用杠杆
- [ ] 保持 6 个月生活开销的现金
- [ ] 每季检查组合的永久性损失风险
- [ ] 极端压力测试：如果所有资产下跌 50%，我还能生活吗？
```
