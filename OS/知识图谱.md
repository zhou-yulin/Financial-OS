# 知识图谱

## 概述
Project Atlas 的知识网络——所有概念、模型、案例之间的连接关系。

## 核心概念节点

### 节点 1：长期主义
```
连接：
长期主义 → 复利思维（时间和复利是长期主义的根基）
长期主义 → 概率思维（长期才能看到概率回归）
长期主义 → 巴菲特（伯克希尔案例的核心理念）
长期主义 → Apple（Apple 的生态建设是长期主义的体现）
长期主义 → 2008 金融危机（危机后的坚持）

应用的 OS：Decision-OS, Life-OS
关联模型：复利思维、机会成本
关联案例：伯克希尔哈撒韦、Amazon
```

### 节点 2：护城河
```
连接：
护城河 → 品牌（品牌溢价是最深的护城河）
护城河 → 转换成本（离开的成本）
护城河 → 网络效应（用户越多价值越大）
护城河 → 规模效应（越大越强）
护城河 → Apple（品牌+生态锁定）
护城河 → Nvidia（CUDA 生态锁定）
护城河 → 茅台（品牌溢价）
护城河 → Costco（会员锁定）

应用的 OS：Business-OS, Financial-OS
关联模型：企业模型、竞争模型、网络效应
```

### 节点 3：复利
```
连接：
复利 → 时间（时间是指数的燃料）
复利 → 财富模型（财富积累的核心机制）
复利 → 长期主义（复利需要时间）
复利 → Amazon（飞轮效应 = 商业复利）
复利 → 伯克希尔哈撒韦（巴菲特 60 年的复利实践）

应用的 OS：Financial-OS, Life-OS
关联模型：财富模型、飞轮模型
```

### 节点 4：决策
```
连接：
决策 → Decision-OS（中枢处理器）
决策 → 概率思维（不确定性下的决策）
决策 → 信号与噪音（减少信息干扰）
决策 → 行为金融（克服认知偏误）
决策 → 人生决策模板（重大选择的工具）
决策 → 机会成本（每个选择的真正代价）

应用的 OS：Decision-OS（中枢）
关联模型：概率思维、机会成本、信号与噪音、行为金融
关联模板：人生决策模板
```

### 节点 5：创新
```
连接：
创新 → S-Curve（创新的生命周期）
创新 → 创新模型（颠覆式 vs 渐进式）
创新 → 竞争模型（创新的竞争含义）
创新 → Tesla（颠覆式创新案例）
创新 → Nvidia（CUDA 开创通用计算）
创新 → AI（范式转移）
创新 → 字节跳动（AI 原生的创新）

应用的 OS：Business-OS, Creator-OS
关联模型：S-Curve、创新模型、竞争模型
```
