# AI 项目分析模板

> 用于评估 AI 赛道中的公司、产品或投资机会。

---

| 项目 | 内容 |
|------|------|

---

## 第一部分：AI 能力评估

### 技术壁垒

```
- [ ] 核心技术是什么？（算法、数据、算力？）
- [ ] 技术领先程度？竞争者需要多久追上？
- [ ] 数据飞轮存在吗？（用户越多 → 数据越多 → 产品越好）
- [ ] 对第三方模型的依赖程度？
```

### 数据护城河

```
- [ ] 是否有独占数据源？
- [ ] 数据在持续增长吗？
- [ ] 数据的边际价值是否递增？（更多数据是否真的带来更好结果？）
- [ ] 用户产生的数据是否可被竞争者复制？
```

## 第二部分：商业模式

### 价值捕获

```
- [ ] AI 如何转化为收入？
  □ API 调用收费  □ 订阅制  □ 广告  □ 企业合同  □ 其他
- [ ] 毛利率预期：_____%
- [ ] 推理成本是否随着规模降低？
- [ ] 是否面临商品化风险？（AI 能力逐渐趋同）
```

### 竞争位置

```
- [ ] 在整个 AI 价值栈中处于什么位置？
  ┌─ 芯片/算力
  ├─ 云基础设施
  ├─ 基础模型
  ├─ 中间件/工具
  ├─ 垂直应用
  └─ 消费级产品
- [ ] 这个位置的价值捕获能力如何？
```

---

## 第三部分：AI 特有风险

```
- [ ] 模型被替代的风险（下一代模型是否会让现有模型过时？）
- [ ] 开源模型的威胁（Meta Llama、Mistral 等是否在蚕食市场？）
- [ ] 监管风险（AI 监管政策是否可能影响业务？）
- [ ] 算力依赖（是否过于依赖 Nvidia 等供应商？）
- [ ] 巨头的挤压（Google、Microsoft、Amazon 是否会进入这个细分市场？）
- [ ] 数据隐私问题
- [ ] 幻觉/安全性问题
```

---

## 第四部分：投资判断

### 关键问题

```
1. 这家公司是在"卖铲子"还是"淘金"？
   （卖铲子 = AI 基础设施，更高确定性）

2. 如果 GPT-5/6 免费了，这家公司还存在吗？
   （如果依赖当前模型的能力差距，这个差距可能在缩小）

3. 用户粘性来自 AI 还是来自其他？
   （AI 能力本身粘性低，生态/数据/工作流粘性高）

4. 护城河是暂时的还是持久的？
   （技术领先是暂时的，数据飞轮/网络效应/生态锁定是持久的）
```

### 决策

- 竞争优势来源：□ 技术 □ 数据 □ 生态 □ 品牌 □ 先发
- 竞争优势可持续性（1-5）：_____
- 估值是否合理：□ 低估 □ 合理 □ 高估
- 投资决策：□ 买入 □ 观察 □ 不参与

---

*在 AI 时代，最重要的问题从"技术是否强"变成了"护城河是否深"。*
