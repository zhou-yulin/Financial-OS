## 第一部分：问题定义

### 核心问题

```
- [ ] 你要解决什么问题？
- [ ] 这个问题有多痛？（1-5，5 = 不做这件事会死）
- [ ] 目标用户是谁？有多少人？
- [ ] 他们现在如何解决这个问题？
```

### 问题验证

```
- [ ] 你亲身体验过这个问题吗？
- [ ] 你与 20+ 潜在用户交谈过吗？
- [ ] 用户是否主动在寻找解决方案？
```

---

## 第二部分：解决方案

### 价值主张

一句话描述你的解决方案：_____

### 解决方案特征

```
- [ ] 比现有方案好多少？（10%更好？10倍更好？）
- [ ] 用户为什么选你而不是现有方案？
- [ ] 你的方案是不可逆的吗？（用户用了就回不去？）
```

### 产品-市场匹配信号

- [ ] 用户主动推荐给他人（NPS > 50）
- [ ] 即使产品不完美，用户仍然使用
- [ ] 用户数在没有广告的情况下增长

## 第三部分：市场分析

### 市场规模

| 类型 | 估算 | 说明 |
|------|------|------|
| TAM（总可寻址市场） | | 全球/全国所有潜在用户 |
| SAM（可服务市场） | | 你的能力范围内可触及的 |
| SOM（可获得市场） | | 短期内现实可获取的 |

### 市场特征

```
- [ ] 市场在增长还是萎缩？增速 _____%
- [ ] 市场是集中还是分散？
- [ ] 进入壁垒：_____
- [ ] 有没有明显的网络效应机会？
- [ ] 监管环境是否有利？
```

## 第四部分：商业模式

### 收入模式

- 核心收入来源：_____

- 客单价：_____
- 预期毛利率：_____%

### 单位经济学

```
- CAC（客户获取成本）：_____
- LTV（客户终身价值）：_____
- LTV/CAC 比率：_____（>3 为健康）
- 回本周期：_____ 个月
```

### 规模化潜力

```
- [ ] 是否可以在不增加相应成本的情况下扩大？
- [ ] 扩张瓶颈是什么？
- [ ] 是否存在网络效应？
- [ ] 是否存在规模效应？
```
