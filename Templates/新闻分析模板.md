# 新闻分析模板

> 每天面对海量新闻，这个模板帮你区分噪音和信号。

---

## 新闻分析流程

### 第 1 步：快速过滤

```
- [ ] 这条新闻会影响企业的长期价值吗？
  ├── 是 → 继续分析
  └── 否 → 忽略（可能是噪音）
```

大多数新闻是噪音。只分析会**改变你对企业判断**的新闻。

### 第 2 步：分类

| 类型 | 标志 | 重要性 |
|------|------|--------|
| 商业模式变化 | 新收入来源、业务重组 | ⭐⭐⭐⭐⭐ |
| 竞争格局变化 | 新进入者、替代品 | ⭐⭐⭐⭐ |
| 管理层变动 | CEO/CFO 更换 | ⭐⭐⭐⭐ |
| 宏观/监管 | 新政策、利率变化 | ⭐⭐⭐ |
| 季度业绩 | 超预期或不及预期 | ⭐⭐（短期） |
| 市场情绪 | 分析师观点、评论 | ⭐（噪音） |

### 第 3 步：深度分析

```
## 新闻标题：
## 日期：
## 来源可靠性（1-5）：
## 新闻类型：
## 核心信息（一句话）：

## 我的分析：
- 这是否改变了企业的竞争态势？
- 这是否影响了企业的长期增长前景？
- 市场可能的短期反应 vs 长期实质影响：

## 行动：
- [ ] 需要调整仓位？
- [ ] 需要修改估值假设？
- [ ] 仅关注，无需行动
```

### 第 4 步：追问

```
- [ ] 这条新闻 6 个月后还重要吗？
- [ ] 市场是否过度反应了？
- [ ] 有没有被忽略的利好或利空？
```

## 常见陷阱

| 陷阱 | 应对 |
|------|------|
| 过度反应：把临时问题当永久变化 | 问："6 个月后这还重要吗？" |
| 确认偏误：只看符合自己判断的分析 | 主动找不同观点 |
| 信息过载：分析太多新闻，分散注意力 | 只分析影响长期价值的 |
| 速度陷阱：急于在新闻出来时操作 | 设定 24 小时冷却期 |

---

*最好的投资者不是在消化所有新闻，而是在辨识哪些新闻值得关注。*
