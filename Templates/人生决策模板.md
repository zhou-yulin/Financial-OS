## 第一阶段：澄清价值观

### 核心问题

```
- [ ] 10 年后，这个决定还重要吗？
- [ ] 这个决定会让我 80 岁时感到骄傲还是遗憾？
- [ ] 我是因为渴望（走向更好的未来）还是恐惧（逃避当前的问题）做决定？
- [ ] 如果金钱不是问题，我会怎么选？
```

### 价值观排序

将以下按重要性排序（1 最重要，6 最不重要）：

```
[ ] 自由（时间和选择的自主权）
[ ] 成长（学习和进步的机会）
[ ] 财富（经济回报）
[ ] 关系（与谁在一起）
[ ] 影响（对他人的影响和贡献）
[ ] 稳定（安全感和可预测性）
```

## 第三阶段：认知偏误检查

```
- [ ] 确认偏误：我只在寻找支持我偏好的证据吗？
- [ ] 沉没成本：我是否因为已经投入而不敢改变？
- [ ] 损失厌恶：我是否过度放大了可能的损失？
- [ ] 现状偏误：我是否仅仅因为"不想改变"而拒绝更好的选项？
- [ ] 社会认同：我是否因为"别人都这样"而影响判断？
- [ ] 近因偏误：最近发生的事是否过度影响了我的判断？
```
